# 2. API Routes/服务端写接口

1. Next.js 提供了编写后端接口的功能（即 API Routes），`编写接口可以在 pages/api 目录下编写`
2. 在 pages/api 目录下的任何 API Routes `文件都会自动映射到以 /api/* 前缀开头接口地址`
3. 比如：编写一个 /api/user 接口; [代码](pages/api/login.ts)
    1.在 pages/api 目录下新建 user.ts
    2.接在在该文件中使用 `handler(约定)` 函数来定义接口
    3.然后就可以用 fetch 函数 或 axios 轻松调用： /api/user 接口了
