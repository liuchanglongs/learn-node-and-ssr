### loading.tsx

特殊文件 loading.tsx 可帮助你使用React Suspense 创建有意义的加载 UI。通过此约定，你可以在加载路由段内容时显示来自服务器的即时加载状态。渲染完成后，新内容会自动换入。

> 当前层级没有loading.tsx，就会寻找父级的

### not-found.tsx

- 当 notFound 函数在路由段中引发时，not-found 文件用于呈现 UI。
- 也可以在 /app 下添加全局的not-found文件：路由有没有匹配会触发、或者调用notFound

```js
import { notFound } from "next/navigation";
notFound();
```
