### 服务端处理
将React组件树渲染为一个HTML字符串或管道化的Node.js流。

+ renderToString
+ renderToPipeableStream

### 客户端处理
对服务端渲染的HTML结构，进行交互处理。

+ hydrateRoot

> 原则：为了保证服务端和客户端能够完美的进行配合，需要React组件在服务端和客户端保持一致，即同构。
>



### 环境搭建
采用create-vite脚手架，具备编译React的环境和SSR辅助方法，默认create-vite搭建的react环境是CSR模式。

```shell
npm create vite@latest
```
