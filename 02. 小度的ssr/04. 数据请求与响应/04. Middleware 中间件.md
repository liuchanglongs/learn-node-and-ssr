### 概述

中间件允许你在**请求完成之前运行代码**。然后，根据传入的请求，你可以通过重写、重定向、修改请求或响应标头或直接响应来修改响应。

在src目录下定义文件，middleware.ts，基本格式如下：

```typescript
import { NextResponse, NextRequest } from "next/server";
export function middleware(request: NextRequest) {
  return NextResponse.next();
}
```

### 匹配路径

> 匹配到的路径都会被拦截

```typescript
export const config = {
  matcher: ["/api/:path*", "/order/:path*"],
};
```
