

### 基本操作
```typescript
async function createAction(formData: FormData) {
  'use server'
}
<form action={createAction}></form>
```



### 携带更多参数
方案一：通过`<input type="hidden" name="userId" value={userId} />`进行传递

方案二：通过`const createActionWithId = createAction.bind(null, userId)`进行传递

:::info
注意：客户端组件的form表单action属性，可以接收异步函数也可以接收同步函数。

:::



### 编程方式触发表单提交
```typescript
const handleKeyDown = (e: React.KeyboardEvent<HTMLInputElement>) => {
  if (e.key === 'Enter') {
    e.preventDefault()
    e.currentTarget.form?.requestSubmit()
  }
}
<input className="border" type="text" name="username" onKeyDown={handleKeyDown} />
```
