

### 概述
Server Actions（服务器操作）<font style="color:rgb(23, 23, 23);">是在服务器上执行的异步函数。可以在 Server 和 Client Components 中调用它们，以处理 Next.js 应用程序中的表单提交和数据更改。</font>

<font style="color:rgb(23, 23, 23);"></font>

<font style="color:rgb(23, 23, 23);">好处：</font>

+ 减少fetch和路由处理程序的调用
+ 与表单结合非常的灵活

<font style="color:rgb(23, 23, 23);"></font>

<font style="color:rgb(23, 23, 23);">服务器组件：内联的形式只有服务器组件可以使用</font>

```typescript
export default async function Page() {
  async function createAction() {
    'use server'
    return 'create action'
  }
  return (
    <div>
      ...
    </div>
  )
}
```



客户端组件：<font style="color:rgb(23, 23, 23);">要在客户端组件中调用服务器操作，请创建一个新文件并在其顶部添加</font>`<font style="color:rgb(23, 23, 23);background-color:rgb(247, 247, 247);">“use server”</font>`<font style="color:rgb(23, 23, 23);">指令。</font>

```typescript
import { createAction } from "./actions"
```

:::info
注意：server action必须是一个async异步函数

:::



### <font style="color:rgb(28, 30, 33);">将操作作为 props 传递</font>
<font style="color:rgb(28, 30, 33);">你还可以将服务器操作作为属性传递给客户端组件（没有use server的函数不能传递）：</font>

```typescript
<ClientComponent createAction={createAction} />
```
