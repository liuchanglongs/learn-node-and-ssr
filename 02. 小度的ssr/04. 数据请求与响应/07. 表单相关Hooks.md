## React19新Hooks

新的Hooks包括：useActionState、useFormStatus、useOptimistic。

- `useActionState`，它允许你根据表单操作的结果来更新 state。
- `useFormStatus`，它为你提供上次表单提交的状态信息。
  > https://react.dev/reference/react-dom/hooks/useFormStatus
  > 表单提交期间显示待处理状态
  > 阅读提交的表单数据
- `useOptimistic` ，可以让你乐观地更新 UI。

```js
setOptimisticMessages((messages) => [...messages, message]);
const sendMessage = await createAction(message);
setMessages((messages) => [...messages, sendMessage]);
对比;
setMessages((messages) => [...messages, message]); // ① 先追加用户输入
const sendMessage = await createAction(message); // ② 等 2s
setMessages((messages) => [...messages, sendMessage]); // ③ 再追加服务端结果
```

> useOptimistic 的作用是：真实列表 messages 在请求完成前不变；只有展示层先多一条；等服务端结果到了，用一次 setMessages 写入真相，乐观层自动对齐，不用手写「先加再删再改」
