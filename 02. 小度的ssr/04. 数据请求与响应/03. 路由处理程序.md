

### 概述
<font style="color:rgb(23, 23, 23);">路由处理程序允许您使用 Web </font>请求<font style="color:rgb(23, 23, 23);">和</font>响应<font style="color:rgb(23, 23, 23);"> API 为给定路由创建自定义请求处理程序。</font>

<font style="color:rgb(23, 23, 23);"></font>

<font style="color:rgb(23, 23, 23);">支持以下 </font>HTTP 方法<font style="color:rgb(23, 23, 23);">：</font>`<font style="color:rgb(23, 23, 23);background-color:rgb(247, 247, 247);">GET</font>`<font style="color:rgb(23, 23, 23);">、</font>`<font style="color:rgb(23, 23, 23);background-color:rgb(247, 247, 247);">POST、</font>``<font style="color:rgb(23, 23, 23);background-color:rgb(247, 247, 247);">PUT</font>`<font style="color:rgb(23, 23, 23);">、</font>`<font style="color:rgb(23, 23, 23);background-color:rgb(247, 247, 247);">PATCH、</font>``<font style="color:rgb(23, 23, 23);background-color:rgb(247, 247, 247);">DELETE、</font>``<font style="color:rgb(23, 23, 23);background-color:rgb(247, 247, 247);">HEAD</font>`<font style="color:rgb(23, 23, 23);"> 和 </font>`<font style="color:rgb(23, 23, 23);background-color:rgb(247, 247, 247);">OPTIONS</font>`<font style="color:rgb(23, 23, 23);">。如果调用了不受支持的方法，Next.js将返回 </font>`<font style="color:rgb(23, 23, 23);background-color:rgb(247, 247, 247);">405 Method Not Allowed</font>`<font style="color:rgb(23, 23, 23);"> 响应。</font>

<font style="color:rgb(23, 23, 23);"></font>

<font style="color:rgb(23, 23, 23);">除了支持原生</font>请求<font style="color:rgb(23, 23, 23);">和</font>响应<font style="color:rgb(23, 23, 23);"> API 之外，Next.js 还通过 </font>NextRequest<font style="color:rgb(23, 23, 23);"> 和 </font>NextResponse<font style="color:rgb(23, 23, 23);"> 对其进行了扩展，以便为高级使用案例提供方便的帮助程序。</font>



好处：

+ 自定义请求处理
+ 简单的mock数据
+ 直接调用数据库
+ 对请求进行统一拦截

### 参数处理

// GET DELETE
const searchParams = new URL(request.url).searchParams

// POST PUT
const requestJson = await request.json()
