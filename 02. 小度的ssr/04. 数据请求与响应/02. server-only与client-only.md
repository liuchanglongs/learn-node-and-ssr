

### server-only
代码只能在服务端使用，不能在客户端使用。

```bash
npm i server-only
```



### client-only
代码只能在客户端使用，不能在服务端使用。

```bash
npm i server-only
```



隐私数据可通过 experimental_taintUniqueValue 防止泄露

```typescript
import { experimental_taintUniqueValue } from 'react'
experimental_taintUniqueValue(
  "Do not pass the key to the client",
  data,
  data.key
)
```
