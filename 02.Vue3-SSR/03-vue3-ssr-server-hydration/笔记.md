# Hydration

服务器端渲染页面 + 客户端激活页面，是页面有交互效果（这个过程称为：Hydration 水合）
◼ Hydration 的具体步骤如下：
 1.开发一个 App 应用，比如 App.vue
 2.将 App.vue 打包为一个客户端的 client_bundle.js 文件
✓ 用来激活应用，使页面有交互效果
 3.将 App.vue 打包为一个服务器端的 server_bundle.js 文件
✓ 用来在服务器端动态生成页面的 HTML
 4. server_bundle.js 渲染的页面 + client_bundle.js 文件进行 Hydration

- [服务端渲染代码](src/server/index.js)
  > 浏览器渲染的时候会，会去脚本 HTML 中的字符串脚本：`<script src="/client/client_bundle.js"></script>`使页面有交互效果，这个时候需要服务器开启静态资源访问路径。
  >
  > - server.use(express.static("build"))：给 Express 服务器挂载一个“静态资源中间件”，把 build 目录作为静态文件目录对外提供访问。
